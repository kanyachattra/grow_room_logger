วิธีติดตั้ง (คลิกเพื่อขยาย)
1
สร้าง Google Sheets ใหม่
sheets.google.com → ไฟล์ใหม่ → ตั้งชื่อ "Grow Room Logger"
2
เปิด Apps Script
ส่วนขยาย → Apps Script → ลบ code เดิมทั้งหมด → วาง code จากไฟล์ GrowRoom_AppScript.js
3
รัน initSheets()
กดปุ่ม ▶ Run เลือก initSheets → อนุญาต Permission → Sheet จะถูกสร้างอัตโนมัติ
4
Deploy เป็น Web App
Deploy → New Deployment → Web App
Execute as: Me
Access: Anyone
→ คัดลอก URL
5
วาง URL ด้านบน
แล้วกด "ทดสอบการเชื่อมต่อ" ✓
6
เชื่อม AppSheet
appsheet.com → New App → Google Sheets → เลือก Spreadsheet เดียวกัน → AppSheet สร้าง View ให้อัตโนมัติ
7
LINE Notify
notify.line.me → Login → Generate token → เลือกกลุ่ม → วาง Token ในแท็บ 💬 LINE
